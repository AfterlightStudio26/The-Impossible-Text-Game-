# The Impossible Text

A pixel horror game made by **Afterlight Studio**.

## About

You receive a text message from a number that doesn't exist.

At first it seems like a normal conversation, but the messages become stranger. The person on the other side knows things they shouldn't know.

Your choices decide what happens next.

## Current Version

`v0.1.0`

The game is still in development.

## Features

* Pixel-style phone interface
* Choice-based dialogue
* Horror mystery story
* Multiple endings (planned)
